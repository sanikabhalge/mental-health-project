# MindCare – Mental Health Chat Project

Frontend chat app (React + Vite). Login, Sign up, Chat with sidebar, camera/mic toggles, risk-detection placeholder, and chat history in localStorage.

## Run the app

```bash
cd frontend/client
npm install
npm run dev
```

Open http://localhost:5173

## Teammate: clone and run

```bash
git clone https://github.com/YOUR_USERNAME/mental-health-project.git
cd mental-health-project
cd frontend/client
npm install
npm run dev
```

Replace `YOUR_USERNAME` with the repo owner’s GitHub username.
